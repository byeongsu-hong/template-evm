# EVM Template

## Prerequisites

- Git
- Node & Yarn
- [Foundry](https://book.getfoundry.sh/getting-started/installation)

## Initial Setup

- Clone the repo
- `yarn install`
- `forge soldeer install -d`

## Build, Test, Coverage

- `forge build`
- `forge test`
- `yarn coverage`

## Linting

- `yarn lint`
- `yarn lint:check`
